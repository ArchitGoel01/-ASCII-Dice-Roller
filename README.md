# 🎲 ASCII Dice Roller

A simple Python project that simulates rolling multiple six-sided dice using ASCII art. The program allows users to choose how many dice they want to roll, generates random values, displays each die side-by-side in the terminal, and calculates the total score.

---

## 📌 Features

- 🎲 Roll any number of six-sided dice.
- 🎨 Display dice faces using ASCII art.
- 🔢 Generate random dice values.
- ➕ Calculate the total of all dice rolled.
- 💻 Simple and beginner-friendly Python project.

## ▶️ How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/ASCII-Dice-Roller.git
```

2. Navigate to the project folder:

```bash
cd ASCII-Dice-Roller
```

3. Run the program:

```bash
python dice_roller.py
```

4. Enter the number of dice you want to roll.
